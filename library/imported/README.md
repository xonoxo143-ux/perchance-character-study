# Imported Characters

Use this folder for characters imported from outside sources.

Examples:

- characters from copied repos
- public Perchance-compatible examples
- character packs
- third-party cards converted for ACC study
- imported characters we want to inspect before rebuilding

## Rule

Imported does not mean trusted.

Each imported character should be treated as study material until evaluated.

## Suggested layout

```text
library/imported/characters/
  character-name/
    character.jsonc
    notes.md
```

## Notes template

```text
Name:
Source:
License / reuse status:
Why imported:
Observed strengths:
Observed risks:
ACC compatibility notes:
Build lessons:
```
