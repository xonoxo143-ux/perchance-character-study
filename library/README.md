# Character Library

This is the front room of the repo: the place to look for characters.

The rest of the repo is the lab. The library should stay readable, navigable, and useful without requiring the reader to understand every ACC test or source note.

## Main index

Start here:

```text
library/character-index.md
```

The index tracks character names, source, status, purpose, quality, and notes.

## Sections

### `imported/`

Characters imported from outside sources, public repos, examples, packs, or existing Perchance-compatible material.

Use this for characters we did not originally build, even if we later study or modify them.

### `made/`

Characters built deliberately in this project.

Use this for stable or semi-stable characters we create from the build/foundation workflow.

### `experiments/`

Unstable characters, test builds, prototypes, layer tests, and deliberately weird constructions.

Use this for characters that teach us something but are not ready to be treated as usable builds.

## Rule

Do not turn the library into a dump.

Every character should have an index entry explaining why it is here.
