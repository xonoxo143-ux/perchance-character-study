# Made Characters

Use this folder for characters built deliberately inside this project.

These are characters created through the Build Lane + Foundation Lane workflow.

## Rule

A made character should include enough notes that another assistant can understand why it is built the way it is.

## Suggested layout

```text
library/made/characters/
  character-name/
    character.jsonc
    build-notes.md
    test-notes.md
```

## Build notes template

```text
Name:
Goal:
Core identity:
Role/reminder/GWI split:
Lore strategy:
Initial-message strategy:
Known risks:
Test prompts:
Current quality:
Next fix:
```
